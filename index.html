READ FIRST — DATA WARNING

Your workout data lives ONLY in this browser.

If you clear your browser data, switch devices, or uninstall this app, everything is gone. There is no cloud backup. There is no account.

What you must do:

  • Visit DATA + SETNG → BACKUP & RESTORE regularly
  • Hit EXPORT — it downloads a CSV file with all your data
  • Save that CSV somewhere off this device (cloud storage, email it to yourself, anywhere)
  • Do this at least once a week
  • If you lose your data, IMPORT the CSV brings everything back
